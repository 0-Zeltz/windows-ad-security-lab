# Windows AD Security Lab

## Overview

This lab simulates a small corporate Windows environment with centralized authentication, security monitoring, and endpoint protection.

---

## Objectives

- Deploy Windows Active Directory  
- Create directory objects  
- Join machines to the domain  
- Apply GPOs by group  
- Implement Wazuh SIEM/XDR  
- Simulate basic attacks and detections  

---

## Infrastructure

- OPNsense Firewall  
- Windows 11 Pro (2x)  
- Ubuntu Server / Apache Server  
- Windows Server (Domain Controller)  
- Wazuh Agent (SIEM & XDR)
